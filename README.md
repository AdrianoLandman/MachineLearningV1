This repository contains 3 different files and each file corresponds with questions 2a, 2b and 2c. Each of the files in the main branch can be run seperately to give the necessary values that are asked for in the assignment.

File 2a_self_code gives the optimized constant velocity values in the x, y and z direction. Furthermore, it gives the final residual error (Sum of Squared Errors). It is explained in text in the code where the input values for the gradient solver can be changed, like the learning rate, maximum number of iterations or the tolerance.

File 2b_self_code gives the optimized constant velocity values and the optimized constant acceleration values in the x, y and z direction. Furthermore, it gives the final residual error (Sum of Squared Errors). It is explained in text in the code where the input values for the gradient solver can be changed, like the learning rate, maximum number of iterations or the tolerance.

The first part of the 2c_self_code is the same as 2c_self_code to make each of the files be able to run separately. This files, additionally, gives the predicted position in T=7. Furthermore, it plots this predicted position with the actual postions from timestep 1 to 6.
